## Overview

Bharat MCP supports multiple output languages. Set `BHARAT_MCP_LANG=hi` to receive all tool responses in Hindi with Indian number formatting.

## Supported Locales

| Locale code | Language | Status |
|---|---|---|
| `en` | English | Stable |
| `hi` | Hindi | Stable |

## Enabling Hindi Mode

### Claude Desktop (stdio)

```json
{
  "mcpServers": {
    "bharat-mcp": {
      "command": "npx",
      "args": ["bharat-mcp"],
      "env": {
        "BHARAT_MCP_LANG": "hi"
      }
    }
  }
}
```

### Docker

```yaml
environment:
  BHARAT_MCP_LANG: hi
```

## What Changes in Hindi Mode

- Field labels translated to Hindi
- Numbers formatted using Indian numbering system (lakhs, crores)
- Status values translated (e.g., `ACTIVE` → `सक्रिय`)
- Summary narratives generated in Hindi
- Error messages in Hindi

### Example

English output:
```json
{
  "status": "ACTIVE",
  "legalName": "FLIPKART INDIA PRIVATE LIMITED",
  "state": "Maharashtra",
  "authorisedCapital": 10000000
}
```

Hindi output:
```json
{
  "स्थिति": "सक्रिय",
  "कानूनी नाम": "फ्लिपकार्ट इंडिया प्राइवेट लिमिटेड",
  "राज्य": "महाराष्ट्र",
  "अधिकृत पूंजी": "1 करोड़"
}
```

## Adding a New Language

1. Create a locale file at `src/i18n/locales/{lang}.json`
2. Copy `en.json` as a template and translate all keys
3. Register the locale in `src/i18n/index.ts`
4. Add the new `lang` value to the `BHARAT_MCP_LANG` enum in `src/core/config.ts`
5. Submit a PR — see [Contributing](/community/contributing)

## Number Formatting

Indian number formatting is applied to all monetary and large numeric fields:

| Value | English | Hindi |
|---|---|---|
| 100,000 | 1,00,000 | 1 लाख |
| 10,000,000 | 1,00,00,000 | 1 करोड़ |
| 1,000,000,000 | 1,00,00,00,000 | 1 अरब |
