# भारत MCP

**India's Business Brain for AI** is a Model Context Protocol (MCP) server that gives AI assistants like Claude real-time access to Indian business data — GST registrations, HSN codes, company filings, director networks, and more.

## What is Bharat MCP?

Bharat MCP exposes a suite of tools over the MCP protocol so that any compatible AI client can:

- **Verify GST registrations** in real time against the GSTN database
- **Look up HSN/SAC codes** and applicable tax rates
- **Fetch MCA company data** — CIN, directors, compliance status
- **Resolve entities** across GST and MCA (GSTIN → PAN → CIN)
- **Score vendor risk** using multi-signal analysis

All responses are returned in structured JSON, with optional Hindi summaries.

## Who is it for?

| Persona | Use case |
|---|---|
| CA / Tax Consultant | Verify GSTINs, check return filing status, compute GST on invoices |

```

  Bharat MCP Server
```
