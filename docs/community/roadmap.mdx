## Near Term (v1.1)

- **E-way Bill tools** — Generate and verify e-way bills via NIC API
- **PAN verification** — Direct PAN status check against NSDL
- **UPI business lookup** — Resolve UPI VPAs to registered business names
- **FSSAI lookup** — Food business operator licence verification
- **MSME/Udyam verification** — Udyam registration status by number

## Medium Term (v1.2)

- **Tamil, Telugu, Kannada** language support
- **Webhook mode** — Push notifications for GSTIN status changes
- **Scheduled monitoring** — Set up alerts for vendor compliance changes
- **CSV/Excel batch import** — Drop a spreadsheet, get enriched output
- **OpenAPI spec** — REST API as an alternative to MCP transport

## Long Term

- **EPFO/ESIC verification** — Employee provident fund registration check
- **Import/Export code lookup** — IEC validation via DGFT
- **Court record integration** — NCLT case lookups for company litigation
- **Credit bureau signals** — Integration with CIBIL / Equifax (via partner API)
- **Regional language OCR** — Extract GSTIN/PAN from scanned documents
