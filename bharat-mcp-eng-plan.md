## Key Interfaces

```typescript
// Adapter interface (core/src/types.ts)
interface DataAdapter<TQuery, TResult> {
  name: string
  fetch(query: TQuery): Promise<TResult>
  healthCheck(): Promise<boolean>
}

// Adapter chain (fallback pattern)
class AdapterChain<T,R> {
  constructor(adapters: DataAdapter<T,R>[])
  async fetch(query: T): Promise<R & { _meta: SourceMeta }>
}

// Response metadata (every tool response includes this)
interface SourceMeta {
  adapter: string       // "mock" | "sandbox" | "gstn-public" | "api-setu" | "gsp"
  cached: boolean
  cachedAt?: string     // ISO 8601
  fetchedAt: string
  latencyMs: number
  partial: boolean
  locale: "en" | "hi"
}

// DI interfaces for entity resolution
interface GSTDataProvider {
  verifyGSTIN(gstin: string): Promise<GSTINResult>
  getReturnStatus(gstin: string, fy?: string): Promise<ReturnStatus>
}
interface MCADataProvider {
  lookupByPAN(pan: string): Promise<CompanyResult[]>
  getComplianceStatus(cin: string): Promise<ComplianceResult>
}

// BaseTool pattern (all 10 tools extend this)
abstract class BaseTool<TInput, TOutput> {
  constructor(cache, adapterChain, i18n)
  async execute(rawInput): Promise<ToolResponse<TOutput>>
  abstract validate(input): TInput
  abstract cacheKey(input): string
  abstract get ttl(): number
}
```

## Error Hierarchy

```
BharatMCPError (base)
├── UpstreamError (government API failures)
│   ├── TimeoutError
│   ├── RateLimitError (retryable)
│   ├── CaptchaBlockedError
│   ├── MalformedResponseError
│   └── AuthError (retryable, token refresh)
├── ValidationError (input validation)
├── EntityNotFoundError (valid input, no match)
├── PartialDataError (some fields unavailable)
└── BatchError (partial batch failure)
```

## Security Notes
- NEVER log raw PAN — use redacted format (XX****XXXX)
- Adapter base URLs are hardcoded/allowlisted (no SSRF risk)
- Devanagari numeral conversion in validation layer
- All responses include source attribution + timestamp
